Request Header Parser API

by QuanchiFootball


This is a simple API that will take an XMLHttp Request and return an object containing 1) the requester's public IP address, 2) the first acceptable response language, and 3) the requester's operating software.


Example

Possible URL Inputs :

https://qcfb-requestheaderparser.glitch.me/whoami

Returns :

{
"ipaddress":"24.80.111.3",
"language":"en-US",
"os":"Windows NT 10.0; Win64; x64"
}
I wrote this API as part of FreeCodeCamp's Back-end development curriculum.
